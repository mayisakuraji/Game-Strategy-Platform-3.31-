4.1 数据库逻辑结构设计

表 4-1 用户信息表 (user)
字段名 数据类型 长度 主键 是否为空 描述
id bigint 20 是 否 用户唯一标识ID
username varchar 50 否 否 用户账号
password varchar 100 否 否 加密密码
nickname varchar 50 否 是 用户昵称
avatar varchar 255 否 是 头像URL
role varchar 20 否 否 角色权限
create_time datetime - 否 是 注册时间
表 4-2 游戏攻略表 (strategy)
字段名 数据类型 长度 主键 是否为空 描述
id bigint 20 是 否 攻略ID
title varchar 100 否 否 攻略标题
game_tag varchar 50 否 否 游戏标签
content longtext - 否 否 攻略正文
cover_image varchar 255 否 是 封面图
author_id bigint 20 否 否 作者ID
status int 1 否 否 审核状态
view_count int 11 否 否 浏览量
create_time datetime - 否 是 发布时间
表 4-3 游戏分类表 (category)
字段名 数据类型 长度 主键 是否为空 描述
id bigint 20 是 否 分类ID
name varchar 50 否 否 分类名称
icon varchar 255 否 是 图标URL
create_time datetime - 否 是 创建时间
表 4-4 评论表 (comment)
字段名 数据类型 长度 主键 是否为空 描述
id bigint 20 是 否 评论ID
strategy_id bigint 20 否 否 攻略ID
user_id bigint 20 否 否 用户ID
content varchar 500 否 否 评论内容
create_time datetime - 否 是 评论时间
